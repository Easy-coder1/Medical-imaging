## Recording Tips

1. **Clear your browser cache** before recording so forms are empty
2. **Use a sample medical image** (you can download one from open sources like Radiopaedia)
3. **Record at 1920x1080** for the best quality
4. **Speak clearly** and leave 1–2 seconds of silence between scenes for editing
5. **Export as H.264 MP4** named `demo.mp4` and place it in the `assets/` folder
